# Sistema de Oficina

Sistema de gerenciamento de oficina mecânica desenvolvido do zero com foco em aprendizado, boas práticas e evolução contínua.

## Objetivo

Aplicar na prática os conhecimentos adquiridos durante minha formação em desenvolvimento de software, utilizando conceitos de programação orientada a objetos, controle de versão, banco de dados e desenvolvimento de sistemas.

## Tecnologias

* Java
* Gradle
* Git
* GitHub

## Status do Projeto

Em desenvolvimento.
